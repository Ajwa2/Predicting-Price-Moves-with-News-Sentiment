# Predicting Price Moves with News Sentiment

This repository contains the project scaffold: editor settings, CI workflow, testing, and basic package layout.

Getting started (Windows PowerShell)

```powershell
python -m venv .venv; .\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m pytest -q
```

Folders

- `.vscode/` — editor settings
- `.github/workflows/` — CI workflows
- `src/` — package source
- `notebooks/` — exploratory notebooks
- `tests/` — unit tests
- `scripts/` — utility scripts

If you need a totally empty repo, the previous history is saved in the local branch `backup-before-wipe`.
